Using multiple Generative AI tools to create a product advertisement video

1: ChatGPT: Generated the entire script using ChatGPT.
2: Stability AI: Used ChatGPT for even generating prompts for creating images on Stability AI.
3: Runway Gen-2: I used these base images to generate video snippets on Runway Gen-2.
4: Soundful: Generated background music on Soundful.
5: Eleven Labs: The voiceover was generated using Eleven Labs. (The lines were generated by ChatGPT only :D)
6: Runway: Compiled everything using Runway's video editor.
